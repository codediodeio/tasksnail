# TaskSnail

TaskSnail is a one-page task management app that uses Rails and Devise.

## Live Demo

[TaskSnail](http://tasksnail.com)

## Parts

- Ruby 2.2.0
- Rails 4.2
- Devise
- Zurb Foundation


## Installation

- `git clone`
- `bundle install`
- `rake db:migrate`
